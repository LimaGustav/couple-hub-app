    │       │

    │   ├── 📂 Data/                    # Persistência de Dados

    │   │   └── 📂 Models/              # ENTIDADES GERADAS PELO SCAFFOLD
    │   │       ├── User.cs             # Classe User (gerada automaticamente)
    │   │       └── Couple.cs           # Classe Couple (gerada automaticamente)
    │   │
    │   └── 📂 Security/                # Utilitários de Segurança
    │       ├── TokenService.cs         # Classe que gera o JWT contendo o UserId e CoupleId
    │       └── PasswordHasher.cs       # Classe para criptografar/verificar senhas (BCrypt/Argon2)
